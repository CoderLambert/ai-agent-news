# 关于本项目

AI Agent News 是一个自动化的 AI Agent 新闻资讯聚合系统，致力于为全球 AI 从业者和爱好者提供及时、全面、多语言的 AI 领域动态。

## 功能特点

- **每小时自动更新**：定时抓取最新新闻，确保信息时效性
- **多源聚合**：整合官方公告、技术博客、社区讨论等多维度内容
- **自动翻译**：非中文内容自动翻译为中文，打破语言壁垒
- **结构化展示**：完整的标题、摘要、原文链接和详情

## 数据来源可靠性说明

我们优先选择以下类型的信息源：

1. **官方渠道**：如 OpenAI News、MIT CSAIL 等，确保信息的准确性和权威性
2. **知名技术博客**：由领域专家撰写，内容经过深思熟虑
3. **主流科技媒体**：拥有专业编辑团队，内容经过审核
4. **活跃技术社区**：通过社区投票和讨论机制筛选高质量内容

请注意：
- 社区讨论内容仅代表作者个人观点，不represent 官方立场
- 部分新闻可能包含推测性内容，建议读者结合多方信息判断
- 我们不对转载内容的真实性负责，请以原文链接为准

## 技术栈

- [VitePress](https://vitepress.dev/) - 静态站点生成器
- Node.js - 新闻抓取脚本
- Cron - 定时任务调度

## 本地运行

```bash
# 启动开发服务器
npm run dev

# 抓取新闻
npm run fetch-news

# 构建生产版本
npm run build
```

## 定时任务

项目配置了每小时自动执行的 cron 任务，自动抓取最新新闻并更新站点。
